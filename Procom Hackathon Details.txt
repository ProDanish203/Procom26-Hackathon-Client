7 Recurrent Bills Tracking - Simplified Features
Core Features:
1. Auto-Detection from Transactions
   * Scan transaction history
   * Identify recurring patterns
   * Detect same merchant/amount monthly
2. Recurrent Bills List
   * Show all detected recurring bills
   * Display: merchant, amount, frequency, next due date
   * Mark bills as "confirmed recurring" or "ignore"
3. Smart Notifications
   * Remind user X days before due date
   * Push notification/SMS/email
   * "Your K-Electric bill is due in 3 days"
4. One-Click Recurring Payment Setup
   * From notification: "Set up auto-pay?"
   * User confirms → auto-pay enabled
   * Choose payment account
   * Set payment date (e.g., 2 days before due)
5. Recurring Payment Dashboard
   * Active auto-pay list
   * Next scheduled payments
   * Pause/cancel auto-pay option
   * Payment history
6. Bill Amount Changes
   * Alert if amount varies significantly
   * "Your electricity bill is Rs. 5,200 this month (usually Rs. 3,800)"
   * User can review before auto-payment
7. Manual Bill Addition
   * Add bills not auto-detected
   * Scan bill image to extract details
   * Set frequency (monthly, quarterly, yearly)
8. Payment Confirmation
   * Notify after auto-payment successful
   * Receipt generation
   * Failed payment alerts
